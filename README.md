# WhatsZap - WhatsApp-like demo

Projeto demonstrativo de um app de chat estilo WhatsApp usando:
- Frontend: React (Vite)
- Backend: Node.js + Express + Socket.IO

## Como rodar localmente

### 1. Clonar
```bash
git clone <SEU_REPO_URL>
cd whatszap_full
```

### 2. Rodar o servidor
```bash
cd server
npm install
npm run dev
```

Servidor por padrão em `http://localhost:4000`.

### 3. Rodar o cliente
```bash
cd ../client
npm install
npm run dev
```

App React por padrão em `http://localhost:5173` (Vite).

## Build / Deploy
- Backend: Railway / Render / Heroku (configure PORT)
- Frontend: Vercel / Netlify
